# The Lineage — Master Reading Curriculum & Tracker

**Goal:** a complete, sequenced self-education in how humanity learned to think, discover, compute, build intelligence, organize civilizations, tell stories, and imagine the future — from Socrates to today's frontier AI.

**How to run it:** work down the spine in order. Finish each module's **Essentials** before moving on; read the *Optional / Companion* titles beside them whenever you have appetite — they're placed where they pay off. Series show **(book 1)**; continue only if hooked. *Research sprints* are 20–40 min bursts (article/lecture/summary), not full books.

**Pace math:** at 3 real hrs/day × 1.5× = **4.5 audiobook-hrs/day**. The 36 essentials ≈ **~480 hrs ≈ ~3.5 months**. Everything (series unexpanded) ≈ 2,100–2,600 hrs ≈ 15–19 months. Hours are approximate audiobook lengths, for pacing only.

### Legend
- **[E]** Essential — the spine (do these first) · **[Opt]** deepening · **[Fic]** companion fiction · **[Fun]** lighter · **[Ref]** reference (dip in)
- `audio` = great on audiobook · `page` = needs eyes (diagrams/proofs/visual)
- ★ = fiction-spine (read to feel the idea)

### The five through-lines
1. **Mechanization of reasoning:** Aristotle → al-Khwarizmi → Leibniz → Boole → Frege → Hilbert → Gödel → Turing → Shannon → von Neumann → McCarthy/Minsky → Hinton/LeCun/Hassabis.
2. **Unification of physics:** Galileo/Kepler → Newton → Faraday/Maxwell → thermodynamics → Einstein → quantum → Standard Model → cosmology.
3. **Taming uncertainty:** Pascal → Bayes/Laplace → Gauss → Fisher → Kahneman → Pearl → Tetlock/Silver.
4. **Nature of mind:** Locke/Hume → Darwin → Simon → Minsky → Hofstadter → neuroscience → modern AI.
5. **How knowledge grows:** Bacon/Descartes → Kuhn → Deutsch → Sagan → complexity.
