Bathroom faucet shower fix:
First thing to try (cheap/easy)
Before replacing it:
• Pull the diverter up and down repeatedly
• Spray vinegar or CLR around the diverter knob/opening
• Let it sit 10–20 minutes
• Run warm water and work it loose
Sometimes hard water buildup is all it is.
If it still sticks
The easiest and most reliable fix is usually:
