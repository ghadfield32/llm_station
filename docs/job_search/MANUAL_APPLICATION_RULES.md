# Manual Application Rules

Route to `Needs Geoff` when any of these appear:

- LinkedIn, Indeed, Workday, Greenhouse, Lever, Ashby, or a custom ATS with
  judgment questions.
- Login, MFA, captcha, bot checks, or session challenges.
- EEO, voluntary self-ID, disability, veteran status, legal certification, or
  background-check acknowledgements.
- Work authorization, sponsorship, clearance, relocation, non-compete,
  start-date, or salary-expectation questions.
- Recruiter or hiring-manager messages.
- Any automation confidence below the configured threshold.

The system may draft suggested wording for ordinary questions, but Geoff reviews
before submitting or sending anything.
