# Code of Conduct

This project follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful and constructive. Harassment, personal attacks, and discriminatory
language are not welcome. Maintainers may remove comments, commits, or contributions
that violate this, and may block repeat offenders.

Report concerns to **ghadfield32@gmail.com**.
